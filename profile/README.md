# Hi there 👋

Duck Organization is an organization that mostly focuses on Opensource projects letting others view & contribute to them freely.

## Contributing

Contributing to Duck Organizations projects can be done by forking the repository you are contributing to and creating a pull request!
Some repositories contain a CONTRIBUTING.md which gives you more detail on how to contribute to that repository.

## Links
